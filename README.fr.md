## 👋 À propos

- 🔭 Focalisé sur les **systèmes full-stack**, l'**IA embarquée** et les **jeux multijoueurs**
- 🔐 **La vie privée est une fonctionnalité, pas un réglage** — si ça peut tourner sur l'appareil, pas besoin du cloud
