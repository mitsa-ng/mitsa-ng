## 👋 About me

- 🔭 Currently focused on **full-stack systems**, **on-device AI** and **multiplayer games**
- 🔐 I believe **privacy is a feature, not a setting** — if it can run on-device, it shouldn't need the cloud
- 🚢 Motto: **ship it, then polish it**
